
# MASTER NET - Site IPTV

## Como publicar
1. Suba esta pasta no GitHub (novo repositório).
2. Conecte o repositório no Vercel.com.
3. Vercel vai detectar e publicar automático.

## Bot
- O bot flutuante usa a imagem `logo-bot.png`.
- Palavras-chave já programadas para respostas automáticas.
- Se não entender a pergunta, redireciona para WhatsApp com a dúvida enviada.

## Comentários
- Só é possível comentar logado com Google.
- O dono do comentário pode apagar o próprio comentário.
